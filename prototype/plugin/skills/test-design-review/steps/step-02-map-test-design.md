# Step 02: Map Test Design

## Goal

Understand what the current test design covers before judging gaps.

## Instructions

1. Extract test areas, scenarios, conditions, and expected results.
2. Identify test design techniques already used:
   - equivalence partitioning
   - boundary value analysis
   - decision table
   - state transition
   - use-case/scenario
   - error guessing
3. Identify explicit oracles:
   - calculation expected value
   - reconciliation check
   - status/permission matrix
   - audit log expectation
4. Mark unclear coverage as unknown.

## Output Fragment

Create a current coverage summary.
