# Step 04: Write Findings

## Goal

Convert coverage gaps into actionable review findings.

## Instructions

Each finding must include:

- ID
- title
- severity
- category
- related viewpoint
- evidence type
- source section
- why it matters
- required change

## Severity Guidance

- Critical: missing coverage can allow security, data integrity, or financial close defects.
- High: missing coverage can allow financial correctness, auditability, or repeated defect patterns.
- Medium: missing coverage affects important but narrower behavior.
- Low: clarification or maintainability issue.

## Output Fragment

Draft the findings section.
