# Step 04: Map Coverage Gaps

## Goal

Identify missing or unknown regression coverage.

## Instructions

1. If existing test cases are provided:
   - map viewpoints to test cases
   - mark coverage as `covered`, `partial`, or `missing`
2. If no test cases are provided:
   - mark coverage as `unknown`
   - list the checks needed to determine coverage
3. Create `missing-regression-tests.md` with suggested tests only when coverage is missing or unknown.

## Output Fragment

Create a coverage matrix:

| Viewpoint | Coverage | Evidence | Gap |
|---|---|---|---|
