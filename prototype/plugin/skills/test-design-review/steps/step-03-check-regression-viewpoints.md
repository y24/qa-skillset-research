# Step 03: Check Regression Viewpoints

## Goal

Compare the test design against selected regression viewpoints.

## Instructions

For each selected viewpoint:

1. Determine whether the test design covers it.
2. Mark coverage as:
   - covered
   - partial
   - missing
   - unknown
3. Identify the missing condition, oracle, data setup, or negative path.
4. Cite source viewpoint IDs and defect pattern IDs.

## Output Fragment

Create a viewpoint coverage matrix:

| Viewpoint | Coverage | Evidence | Gap |
|---|---|---|---|
