# Step 02: Prioritize Patterns

## Goal

Prioritize patterns for regression selection.

## Instructions

1. Use `regression-selection-criteria.md`.
2. Prioritize by:
   - critical or high risk
   - production/escaped defect evidence
   - affected quality characteristic
   - repeat count
   - business criticality
   - change relevance, if change data is available
3. If no change data is available, produce a baseline release regression recommendation.

## Output Fragment

Create a prioritization table:

| Rank | Pattern | Priority | Why | Recommended Cadence |
|---:|---|---|---|---|
