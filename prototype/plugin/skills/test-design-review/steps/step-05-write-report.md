# Step 05: Write Report

## Goal

Produce `test-design-review-report.md`.

## Instructions

1. Write the review summary.
2. Include the viewpoint coverage matrix.
3. List findings ordered by severity.
4. Include open questions.
5. Include recommended test additions.

## Completion Criteria

- Every finding has a related viewpoint or source evidence.
- Every critical/high missing viewpoint has an explicit action.
- The report is useful to a test designer without reading all source files.
